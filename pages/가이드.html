신규가이드 안내

신규분들은 카톡:casper99로 메세지 주시면 신규지원상자 드립니다.

( 1회 한정이며 악용시 벤처리 되니 유의 바랍니다. )

　

유령서버 만인첸트는 " 15무기 / 13방어구 / 9장신구 " 입니다.

　

캐릭생성후 .고정신청 완료시 AC버프 적용 됩니다.

( 추가명령어는 .도움말 입력시 확인 가능합니다. )

　

유령서버는 2개의 클라이언트까지 실행 가능합니다.

( 접속시 10분당 유령접속 상자 지급 ! ( 수표5억 / 하딘의 가넷 10개 )

　

서버오픈후 2일정도는 운영자의선물 지속적으로 지급하니 2클라 필수 입니다.

　

혈맹은 꼭! 가입하시는게 좋으며 혈맹 가입시 많은 지원을 받으실수 있습니다.

　

마을내 허수아비 타격시 " 픽시의 깃털 " 획득 됩니다.

( " 픽시의 깃털 "로 깃털전용 아이템 구입 가능합니다. )

　

캐릭생성후 경험치 지급단 NPC로 90Lv 까지 성장 가능합니다.

( 서버 만레벨은 92 입니다. 추후 만레벨은 조정될수 있습니다. )

　

레벨업으로 지치지 않도록 20위 랭킹물약을 판매합니다.

( 직장인분들은 접속후 랭킹물약복용하시면 레벨차없이 필드 가능 )

　

기본적인 무기 방어구 악세는 1원상점 NPC로 배치되어 있습니다.　

　

특화 방어구 악세는 가넷 전용 상점에서 구입 가능합니다.

　

후원코인은 노가다 가능하며 후원아이템 NPC는 따로 배치되어있습니다.

( 민캐도 2일정도 투자하시면 서버 풀템 가능 합니다. )　

　

잡화 / 버프 / 수표 상인에 각종 소모품 배치되어 있습니다.

　

근/원거리 물약 / 버프코인은 영구사용으로 1개씩 구입하시면 됩니다.

( 근/원거리 물약은 사용시 3억 소모 / 버프코인은 가넷 500개 소모 )

　

사냥터는 켄트던전 24시간 / 2시간 로테이션 사냥터로 편성 되어있습니다.

( 사냥터 정보는 카테고리 사냥터 정보란 확인 부탁드립니다. )

　

메인 보스는 마을내 전용 보스방 NPC로 이동 가능합니다.

( 보스 정보는 카테고리 보스 정보란 확인 부탁드립니다. )

　

서버내 파밍은 로테이션 던전 사냥하시면 가넷 아이템 셋팅 가능합니다.

　

서버 오픈후 2일 간격으로 공성전 진행하며 참여시 공성보상 지급 됩니다.

( 공성 보상은 하딘의 가넷 / 후원코인 이므로 민캐도 쉽게 풀템 가능 )

　

신규유저분들께서 더 궁금한사항은 고객센터 문의 부탁드립니다 !
